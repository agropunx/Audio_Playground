# Live Spectrum Analyser
Simple PyQt Application for visualizing in real time the Waveform(Time Domain) and Spectrum(Frequency Domain) of the 
sound continuously picked by the default system mic.

Two knob-like controls are added to the gui acting as "Gains" for easy control of the visualization.
## Install
run from terminal---> $pip install -r ./requirements.txt

## Usage
simply run from terminal---> $python3 Live_Spectrum_Analyzer.py
